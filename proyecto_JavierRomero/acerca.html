Acerca del Proyecto

InfoTiempo es una aplicación web que permite a los usuarios consultar el estado del clima actual y las previsiones meteorológicas por horas y días, así como mantenerse informados con las noticias más recientes de España.

Objetivos del Desarrollo

  • Mostrar el estado del tiempo actual y futuro de forma clara y sencilla.
  • Ofrecer una interfaz adaptable a móviles y escritorio.
  • Permitir el cambio de tema claro/oscuro.
  • Informar al usuario con noticias relevantes nacionales.

Tecnologías Empleadas

  • HTML5 semántico
  • CSS3 con Flexbox y Grid
  • Bootstrap 5
  • JavaScript puro (sin frameworks)
