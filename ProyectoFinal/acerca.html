Acerca del Proyecto

Descripción General

WheatherApp es una aplicación web diseñada para mostrar información meteorológica actualizada y noticias relacionadas, con un diseño intuitivo y responsivo.

Objetivos del Desarrollo

  • Proporcionar datos meteorológicos precisos y accesibles.
  • Mostrar noticias nacionales relevantes y actuales.
  • Ofrecer una experiencia de usuario sencilla con modo claro y oscuro.
  • Permitir consultar el clima en diferentes ciudades mediante selección dinámica.

APIs Utilizadas

  • WeatherAPI: para obtener datos meteorológicos en tiempo real.

Tecnologías Empleadas

  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript
